# paralyzed-fetch

>It's a example to how you can fetch from an API, for example, and wait for the response to return to previously call using `ForkJoinPool`.


## Installation

### Build
```console
$ mvn clean install
```

### Running
```console
$ mvn spring-boot:run
```

## Usage

### GET /weather/cities
```console
$ curl -X GET http://localhost:8080/weather/cities
```
